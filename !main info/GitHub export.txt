Чтобы обновить файл на GitHub через терминал, выполните следующие команды: git add, чтобы подготовить изменения, git commit, чтобы сохранить их, и git push, чтобы отправить их на удаленный репозиторий. Для начала убедитесь, что вы находитесь в папке своего локального проекта, а затем выполните эти три шага. 
Шаги для обновления файла
Перейдите в каталог вашего проекта

Откройте терминал и используйте команду cd для перехода в корневую папку вашего репозитория.
cd путь/к/вашему/проекту
Подготовьте изменения к фиксации

Используйте команду git add ., чтобы добавить все измененные файлы.
git add .
Зафиксируйте изменения

Используйте команду git commit -m "Сообщение, описывающее изменения" для создания коммита.
git commit -m "Обновлен файл README"
Отправьте изменения на GitHub

Используйте команду git push для отправки коммитов на удаленный репозиторий.
git push
